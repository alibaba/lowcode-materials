# AreaChart

面积图。

使用带填充区域的线段显示数据在一个具有顺序性的维度上的变化。

## API

| 参数   | 说明                                                                               | 类型              | 默认值                                                                                                      | 版本支持 |
| ------ | ---------------------------------------------------------------------------------- | ----------------- | ----------------------------------------------------------------------------------------------------------- | -------- |
| data   | 数据源，为对象集合例如：[{ type: 'a'，value: 20 }, { type: 'b'，value: 20 }]       | Array             | -                                                                                                           |          |
| xField | 图形在 x 方向（横向延伸）对应的数据字段名，一般对应一个连续字段。                  | String            | -                                                                                                           |          |
| yField | 图形在 y 方向对应的数据字段名，一般对应一个离散字段。                              | String            | -                                                                                                           |          |
| color  | 指定图形颜色，即可以指定一系列色值，也可以通过回调函数的方法根据对应数值进行设置。 | String[]/Function | 采用 theme 中的色板。                                                                                       |          |
| label  | 标签文本                                                                           | Object            | `{visible: false,offsetX: 6,offsetY: 6,style:{fill: 'rgba(0, 0, 0, 0.65)',stroke: '#ffffff',lineWidth: 2}}` |          |

完整 API 请参考 [AreaChart](https://bizcharts.alibaba-inc.com/product/BizCharts4/category/77/page/118)
