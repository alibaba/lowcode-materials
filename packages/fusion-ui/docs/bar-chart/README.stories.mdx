# BarChart

条形图。

使用条形显示维度的数值。纵轴显示分类维度，横轴显示相应的值。

## API

| 参数   | 说明                                                                                     | 类型              | 默认值                                                                                                                                                                | 版本支持 |
| ------ | ---------------------------------------------------------------------------------------- | ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| data   | 数据源，为对象集合例如：[{ type: 'a'，value: 20 }, { type: 'b'，value: 20 }]             | Array             | -                                                                                                                                                                     |          |
| xField | 条形在 x 方向长度映射对应的数据字段名，一般对应一个离散字段。                            | String            | -                                                                                                                                                                     |          |
| yField | 条形在 y 方向位置映射所对应的数据字段名，一般对应一个分类字段。                          | String            | -                                                                                                                                                                     |          |
| color  | 条形颜色映射对应的数据字段名。从基础条形图的使用场景来说，我们不推荐进行额外的颜色映射。 | String[]/Function | 采用 theme 中的色板。                                                                                                                                                 |          |
| label  | 标签文本                                                                                 | Object            | `{visible: false, position: 'left',offsetX: 6,offsetY: 6,style:{fill: 'rgba(0, 0, 0, 0.65)',stroke: '#ffffff',lineWidth: 2},adjustColor: true,adjustPosition: false}` |          |

完整 API 请参考 [BarChart](https://bizcharts.alibaba-inc.com/product/BizCharts4/category/77/page/126)
