# PieChart

饼图。

饼图主要用于表示不同分类的占比情况。饼图中的每个扇形切片表现一个分类，扇形切片的弧度表示该分类在整体中的占比，所有切片构成一个整体，即 100%。

饼图是一个饱受争议的图表类型，有人认为饼图看起来难以阅读，我们的视觉系统只能判断出一个切片比一个切片要大，但很难确定大多少。在一些极端的情况下，饼图各切片大小相近，饼图将变得失去其功能性。如果你也有这种担心的话，可以使用条形图代替。

## API

| 参数       | 说明                                                                               | 类型                     | 默认值                | 版本支持 |
| ---------- | ---------------------------------------------------------------------------------- | ------------------------ | --------------------- | -------- |
| data       | 数据源，为对象集合例如：[{ type: 'a'，value: 20 }, { type: 'b'，value: 20 }]       | Array                    | -                     |          |
| angleField | 扇形切片大小（弧度）所对应的数据字段名。                                           | String                   | -                     |          |
| colorField | 扇形颜色映射对应的数据字段名。                                                     | String                   | -                     |          |
| color      | 指定扇形颜色，即可以指定一系列色值，也可以通过回调函数的方法根据对应数值进行设置。 | String/String[]/Function | 采用 theme 中的色板。 |          |
| label      | 标签文本                                                                           | Object                   |                       |          |
| legend     | 图例，配置 colorField 时显示，用于展示颜色分类信息                                 | Object                   |                       |          |

完整 API 请参考 [PieChart](https://bizcharts.alibaba-inc.com/product/BizCharts4/category/77/page/130)
