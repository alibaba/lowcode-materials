# ProDialog

对话框

## 何时使用

- 继承自 fusion 的弹窗，对话框是用于在不离开主路径的情况下，提供用户快速执行简单的操作、确认用户信息或反馈提示的辅助窗口。

## API

### ProDialog

api 继承自[fusion dialog](https://fusion.alibaba-inc.com/pc/component/dialog?themeid=2)

footer 相关的 api 均不可用，需要使用 operations 和 operationConfig 控制

| 参数            | 说明                                                                                          | 类型         | 默认值      | 版本支持 |
| --------------- | --------------------------------------------------------------------------------------------- | ------------ | ----------- | -------- | --- | --- |
| operations      | 定义操作区按钮,tyep 为'ok' 和 'cancel' 支持触发默认操作                                       | ReactArray   | []          |
| operationConfig | 定义操作区域配置, 可配置 fixed?: boolean; // 是否固定;showSaveTime?: boolean;align?: 'center' | 'flex-start' | 'flex-end'; | object   | {}  |     |
